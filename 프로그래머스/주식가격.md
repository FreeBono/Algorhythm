# 주식가격



```python
def solution(prices):
    answer = [0]*len(prices)
      
    
    for j in range(len(prices)):
        cnt = 0
        for i in range(j+1,len(prices)):
            if prices[j] <= prices[i]:
                cnt += 1
            else:
                cnt += 1
                break
        
        
        
        answer[j] = cnt

```
